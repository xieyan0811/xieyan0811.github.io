#Python

一般安装Python的三方库，直接使用Python的包管理工具pip，或者下载源码包后，使用其中的setup.py安装，就可以直接安装在Python的系统库目录中了．

如果想使用一个三方库，又不想安装在Python的默认库目录中，可以程序中使用 ”sys.path.append("具体路径")”
