2. 常用pdb命令

pdb命令和gdb差不多，最常用的命令如下：
(1) 单步调试（进入函数）：s(tep)
(2) 单步调试（不进入函数）：n(ext)
(3) 继续往后执行，直到下个断点：c(ont(inue))
(4) 运行到函数结束：r(eturn)
(5) 运行到当前循环结束：unt(il)
(6) 设置断点：b(reak) 文件名:行号（或行号，或函数名）
(7) 显示当前调用关系：w(here)
(8) 显示当前代码段：l(ist)
(9) 显示变量：p(rint) 变量名
(10) 显示当前函数的参数：a(rgs)
(11) 显示帮助信息：h(elp)
(12) 退出：q(uit)
