Python获取国内股票数据

#python #股票

1 安装支持库

$ pip install panda  
$ pip install tushare

2 说明

Pandas是数据分析工具包
TuShare是国内股票数据抓取工具，除了股票的实时和历史数据，还有基本面数据，加上自然语言处理（比如情绪分析），或者机器学习，就比较有趣了。

3 程序

3.1 代码

import tushare as ts  
d = ts.get_tick_data('601318',date='2017-06-26')  
print d  
e = ts.get_hist_data('601318',start='2017-06-23',end='2017-06-26  
')  
print e
