1 简介

1.1.1 什么是CSS

  • 用途：设置显示风格
  • 英文：Cascading Style Sheets (CSS)
  • 语法：它有自身的语法

1.1.2 优点

  • 准确：更准确地控制显示
  • 省事：一个设置多处使用
  • 适配：在不修改HTML的情况下适配不同显示设备

同样内容，不同风格，看起来有多么不同：

1.2 基本概念

  • 继承：子控件继承父控制属性
  • 冲突：当设置发生冲突时，优先使用权重更大的设置
    • 一般用中括号定义形如：
    p { line-height: 1.2em; }  [0] [0] [1]

1.2.1 基本语法

CSS每一条由两部分组成：
* 第一部分是 selector
* 定义需要被设置的元素。
* 此处可设置多个元素，用逗号分隔
* selector可支持三种类型：
* 选择符（如p），
* 元素的类名（以点开头）
* 元素的 id（以井号开头）
* 用星号定义其它所有情况
* 第二部分是 declaration
* 定义设置的具体规则
* 使用大括号定义
* 每一部分由键值对成，各部分之间使用分号分隔
* 可包含换行

h1 { color: green; }  
li em { color: red; }  // 其中li是em的父控件  
#sleestak { color: olive; }   
.abc { color: olive; }  
p, ul, td, th { color: navy; }  
p  { font-size: large; font-family: sans-serif; }  
* { border: 1px solid gray; }

1.2.2 设置风格的三种方法

方法一 （推荐）

定义在css文件中，被html引用，形如：

<link rel="stylesheet" type="text/css" href="<!--swig￼0-->"/>  
或:  
<style>      
    @import url(external.css);  
</style>
方法二

定义在style标签中，放在html的head里。

<head>  
    <style>  
        /* style rules go here */     
    </style>  
</head>
方法三（不推荐）

定义为标签的style属性。

<h1 style="color: red">Introduction</h1>
