#笔记/obsidian

简介

原来一直觉得对每个小时日程做记录实在太卷了，但是近来常常遇到的问题是：计划都没完成，时间就没了，所以决定记录一下看看每天都做了什么。
Obsidian 的 Day Planner 插件就实现了这一功能，且可以以图的形式显示当天日程。

优点

  • 很直观的看到当前处于什么位置
  • 明显看到时间流浙
  • 哪项工作delay
  • 哪些工作安排冲突
  • 尽管没完成计划，但也并没闲着
  • 进一步优化时间表，更有掌控感

使用方法

  • 安装三方插件Day Planner
    • Setting->Community plugins->Day Planner
    • 这也是一款下载量非常大的插件
  • 设置插件
    • Setting->Day Planner
    • 建议在 Day Planner mode 中选 Command mode，如果使用File mode会建立一个文件夹专门存放Day Planner文件，如果使用Command mode则可通过命令面板在任意文件中插入Day Planner块
    • 建议选中Mermaid Gantt，可以在笔记中自动建立横版的进度图，形如：

  • 添加Day Planner
    • Ctrl+P调出命令面板
    • 输入Day Planner，找到与它相关的所有命令
    • 选 Add a Day Planner...... 在当前笔记中加入基本模板
    • 再选 Show the Day Planer timeline，此时在右侧面板上显示竖版进度图
    • 如果显示“No plan data”，使用命令面板上的 Link today's Day planner to the current note，即可正常显示

  • 关键字
    • 注意关键字 “BREAK” 为休息时段，在横图上它与其它状态以不同颜色显示

Tips

  • 可以把"Kanban"和"Day Planner"放在同一个文件里，个人感觉，这样用也挺不错
