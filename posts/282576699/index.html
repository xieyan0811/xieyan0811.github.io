3D 打印手办上色的材料与优化

Created2026-01-20|Updated2026-01-19|x_3D打印待发
|Word count:1.6k|Reading time:4min|Post View:

做简单上色时，丙烯颜料或丙烯马克笔已经足够；但一旦涉及手办（可收藏的立体模型/人物模型），尤其是脸部，问题立刻显现。人的视觉对脸部非常敏感，这里我们追求的不仅仅是“能上色”，而是细腻、稳定、可控。

普通丙烯在这一步会显得很吃力：表面张力大，一下笔就铺开，画眼线时常见的情况是 0.3 mm 直接变 0.6 mm；干得快，还没推开就结边；一旦糊了，修正基本等于打磨，而打磨约等于毁细节。

这里展示了我踩过的一些坑，结合常见几种材料——丙烯、水彩、色粉、彩铅——推荐一套更稳妥的组合方案。先说结论：先喷消光，大面用丙烯，细部用油性彩铅，上妆和渐变交给色粉（甚至直接用人用彩妆）。

1 底处理（非常重要）

底处理流程很简单：细砂纸打磨 → 清洗 → 喷一层消光。之后每完成一个阶段，都薄喷一次消光，相当于给自己设“存档点”。

1.1 打印横纹的处理

脸部细节一般不建议重度打磨。对小手办来说，五官高度本来就只有 0.3–0.6 mm，一次 800–1000 目砂纸，可能就磨掉 0.05–0.1 mm，代价太大。

横纹最好在打印阶段规避，比如：脸朝上打印，减少层纹穿脸；脸部层高控制在 0.08–0.12 mm。

如果已经有横纹，可在脸部做局部“软填充”：用极稀释的丙烯或补土液，用刷子“扫”，不是涂，只填沟壑即可。补土尽量少用或避开五官，否则细节很容易被吃掉。

可以说：补土是“雕塑工具”，消光是“妆前底漆”。

1.2 消光

PLA / PETG（尤其是 FDM 打印）表面往往带有一层微弱光泽，表面能低，水性丙烯或酒精型马克笔不容易润湿，颜料容易“聚拢”，像水珠落在荷叶上，笔触自然就不顺。

消光漆会在表面形成一层微细哑光纹理，增加微观粗糙度和有效表面积，让颜料和色粉更容易附着、也更牢。若模型表面依然偏滑，可先用 600–800 目砂纸做轻微打磨，再喷消光，效果会更稳定。

消光本质上是“人为制造的微粗糙表面”。当喷得太厚时，消光粉在表面形成大量微孔，液态颜料一接触，水或介质会被毛细作用迅速吸走，于是出现“吃粉、发虚”的情况。放置 12–24 h 后，溶剂继续挥发，树脂进一步成膜，一部分消光粉被包覆，微孔边缘变钝，吸力下降，于是表面“回弹”。这就是为什么很多人经验上都会说：喷完消光，放一晚再画。

我们常遇到的问题是，用丙烯马克笔上色时，需要反复涂两三次才能让颜色“挂得上”并且均匀。而在喷了消光剂之后，用马克笔涂一到两层，就能完美实现上色。

另外，消光有“把颜料固定住”的作用，让树脂渗入并锁住颜料颗粒，因此喷过消光后，颜色也更不容易掉。

2 颜料与工具选择

2.1 水彩

在老派 BJD / GK 圈里，“色粉 + 水彩 + 层层消光”是一条成熟路线。但水性颜料有明显边界，比如：怕喷，第一层消光稍重就可能化；有“自粘性”，稳定性不如丙烯。

因此在细线（睫毛、下眼线）、高饱和实色块、或者不想频繁重喷的场景下，不太推荐水彩。换成模型丙烯、插画丙烯或油性彩铅，会轻松很多。

2.2 丙烯

很多人都会遇到同一个问题：已经喷了消光，但用极细笔画普通丙烯—— 加水太淡，不加水又画不上去。

这不是画功问题，而是材料不匹配造成的：普通丙烯颗粒偏粗，本来就是为铺色设计；加水后颜料密度崩塌，被消光层吸走；不加水则表面张力过大，断线、拉丝。

如果一定要用丙烯，可以这样优化： - 换更细体系的颜料（模型丙烯 / 插画丙烯） - 不加水，改用 Acrylic Flow Improver / Flow Medium 稀释颜料 - 换工具：牙签画下眼线，其实很多人都在用

2.3 彩铅

关键线条——眼线、瞳孔边缘、眉尾——非常适合交给彩铅。原因很简单：它是硬的。
在消光层上，彩铅几乎只参与摩擦行为，不参与流体行为：不晕、不吃、不被吸走，下笔就是结果。

在使用时建议：油性优于水溶（稳定、不怕回潮）；削成长锥形尖，而不是短尖。另外，注意消光不要太“死哑”，否则磨不出色。 一层画完 → 薄喷消光 → 再画 → 再固，使用这个循环，彩铅表现会非常稳定。

2.4 色粉

色粉是累积式的：一层不够再加，不会突然翻车；边界自然，非常适合鼻影、眼窝、唇色这类需要“空气感”的部位。代价只有一个：必须配合消光，否则稳定性不足。

工具建议用软毛眼影刷，橡皮或棉棒修边。色粉也可以混色，而且不是在调色盘上混，而是在脸上叠加，这对皮肤这种非纯色表面尤其友好。

对娃脸来说，常用颜色其实不多（腮红、眼影为主），如果只考虑给娃化妆，买一盘多色彩妆有时候比专业色粉套装更合适：它粒径细（不显“毛孔”）；饱和度适中（不炸色）；色相贴近真实肤色逻辑；容错率高，不容易一刷毁一张脸。

使用时需要注意：一层完成 → 薄喷消光 → 再来一层。

可以说：色粉 + 固色层，是给 BJD 上妆的首选方案。很多看起来很“高级”的脸，真正占面积最大的，其实全是色粉。

3 总结

一句话总结： 先打消光，大面积用丙烯，细线用油性彩铅，渐变和妆感交给色粉。这三种颜料恰好覆盖了：形体 · 线 · 气氛。
