2 国外服务器

2.1 🟥 Hetzner（德国）

  • 产品名称：Hetzner Cloud–CX22
  • 配置：2 vCPU / 4 GB RAM / 40 GB SSD / 20 TB 流量
  • 价格：€4.51 / 月（含税约 $4.85）
  • 年费：约 $58–$60 / 年
  • 备注：性价比极高，适合托管服务、轻量 API、网站等
  • 网址：https://www.hetzner.com/cloud

2.2 🟪 Oracle Cloud（美国）

  • 产品名称：Oracle Cloud Free Tier（Always Free）
  • 配置：最多 2 台 Arm 实例，每台：
    • 4 vCPU / 24 GB RAM（Arm 架构）
    • 200 GB 块存储 + 出站月限 10TB（理论）
  • 价格：永久免费
  • 注意事项：
    • 有额度限制（常因滥用而注册难）
    • 不支持 x86，仅 Arm 架构，对部分软件不兼容
    • 注册容易翻车，但若成功，几乎是白嫖神器
  • 网址：https://www.oracle.com/cloud/free
  • 说明：
    • 使用我的虚拟卡注册，被拒绝，可能由于信用卡信息与住址不一致
    • 使用美国的地址及电话号码，也不行

2.3 🔵 Vultr（美国）

  • 产品名称：Vultr Cloud Compute (VC2) / High Performance
  • 配置：
    • VC2：2 vCPU / 4 GB RAM / 80 GB SSD / 3 TB 流量
    • High Performance 系列略贵，适合 I/O 密集型
  • 价格：$24 / 月
  • 年费：约 $288 / 年
  • 优惠选项：
    • 可选按小时计费（$0.036/hr）
    • 经常发放 $100 免费额度新用户试用
  • 网址：https://www.vultr.com
