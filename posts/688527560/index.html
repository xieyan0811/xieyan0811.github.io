开源项目_大模型应用_awesome-chatgpt-prompts-zh

Created2023-12-14|Updated2025-04-04|2_Note0_Technic0_工具GPT应用增强工具
|Word count:561|Reading time:1min|Post View:

https://github.com/PlexPt/awesome-chatgpt-prompts-zh

1 功能

  • 整体功能，想解决什么问题
    • ChatGPT 中文调教指南：提供一些常用的使用场景及对应的 Prompt 提示
  • 当前解决了什么问题，哪些问题解决不了
    • 针对想解决实际问题，但不知道如何描述，以及不知道 AIGC 可以解决什么问题的用户，进行很好的使用引导。
    • 项目主要提供数据，无代码；程序员可以参考其中的一些提示，普通用户无法直接拿来使用。
  • 提供哪些功能点，其中哪些是刚需
    • 提供丰富的场景与提示的对应关系
  • 用户使用难度，操作逻辑是否过于复杂
    • 作者团队除了提供该应用的网页版，他还提供了 Windows, Mac, Linux, Android，小程序入口。可通过试用和购买其附费服务使用。

2 技术栈

  • 技术栈是什么
    • 这里没有什么代码，主要是场景及相关的提示的数据
  • 现有底层工具消化了哪些常用功能
    • 源自项目：https://github.com/f/awesome-chatgpt-prompts (95.1K star)，可以看作是这个项目的中文版本，但比英文版丰富很多。
  • 代码分析
    • github 下载 135M，其中.git 35M，question 94M
    • 当前版本的 question 中共包含约 67 万个问题

3 商业模式

  • 它不像一般 github 上偏工具类的方法，而更像 AIGC 的辅助工具，操作过程是：用户通过它的树型分类引导，找到自己想要生成的文档类型，简单输入目标，由它调用模型来生成一段文字图片或者语音，然后通过交互来做修改调整。
  • github 可以看作工具的入口，可以跳转到作者的应用：智造喵，以生成周报，生成文案，生成总结，总有几个能打中你的需求。
  • 在项目中确实看到作者团队做了很多的工作。实现的应用明码标价，用户购买服务解决问题。用户省去了搭建，注册，购买多种服务的问题，只需要对他付费，降低了 AIGC 的使用难度。
