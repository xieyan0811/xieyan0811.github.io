1 简介

1.1.1 什么是 CSS

  • 用途：设置显示风格

  • 语法：它有自身的语法

1.1.2 优点

  • 准确：更准确地控制显示
  • 省事：一个设置多处使用
  • 适配：在不修改 HTML 的情况下适配不同显示设备

1.2.1 基本语法

CSS 每一条由两部分组成：

  • 第一部分是 selector
    • 定义需要被设置的元素。
    • 此处可设置多个元素，用逗号分隔
    • selector 可支持三种类型：
      • 选择符（如 p），
      • 元素的类名（以点开头）
      • 元素的 id（以井号开头）
      • 用星号定义其它所有情况
  • 第二部分是 declaration
    • 定义设置的具体规则
    • 使用大括号定义
    • 每一部分由键值对成，各部分之间使用分号分隔
    • 可包含换行
