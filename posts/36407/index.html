介绍

SuperSet 是一款开源可视化 BI（商业智能）Web 应用程序。它通过创建和分享 dashboard，为数据分析提供了轻量级的数据查询和可视化方案。其优点如下：

  • 不需要自己去搭服务，写前端页面，直接生成分析图
  • 可接入多种数据源
  • 安装方便，学习成本较低

使用

  • 增加库

    上方菜单 ->Data->Databases->右上 add DATABASE，此时可加入 MySQL 等各种数据库作为数据源，输入信息后可显示在列表中。

  • 增加表

    上方菜单 ->Data->Datasets->右上角 add DATASET，选库中的表即可即可显示在列表中。

  • 建立 dashboard 仪表盘

    • 上方菜单 ->Dashboard->右上角 add DASHBOARD
    • 按提示建一个 Chart，在该界面选一个表作为数据源，然后选数据表类型，点下方的 CREATE NEW CHART
    • 选择想要展示的列（Query->METRICS），可通过写 SQL 或者选列的方式设置在图中显示的具体内容
    • 点击下方 CREATE_CHART，即可在右侧看到图表
    • 点击右上方保存，即可将其保存到 CHART 及 DASHBOARD 中
