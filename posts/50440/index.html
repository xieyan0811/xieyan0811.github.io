MacOS 入门之一：基本操作

1．说明
本文简述了 Macbook 的一般使用方法，以及 Iphone 开发环境的配置，参考资料和注意事项，希望对初学者有所帮助。

2．基本操作（与非苹果系统差异）

  1. 切换操作系统，开机时按 Option 键

  2. 如何查看系统信息：菜单 ->前往 ->实用工具 ->系统信息

  3. 鼠标操作

  1. 按下左键：单指按下触摸板，触摸板实际是个大按钮

  2. 按下右键：两指按下触摸板

  3. 滚轮：两指划动

  1. 如何弹出光盘：按键盘右上角弹出键，有些光盘正在使用中时，需要右键操作光盘图标，选择推出

  2. 应用退出全屏：全屏时，鼠标放在屏幕顶端，顶端工具条会显示出来，点击工具条最右侧蓝色按钮

  3. 文件管理器：Finder->设备

  4. 修改默认操作系统：菜单 ->苹果图标 ->系统偏好设置 ->启动磁盘

  5. 如何切换输入法：Command+ 空格

3．应用程序

  1. Mac OS 中的常用软件

  1. 播放多媒体工具：mplayer

  2. 聊天工具：QQ

  3. 网络电视：PPTV

  4. 游戏

  5. 常用工具：FlashPlayer

  6. 虚拟机工具：VirtualBox

  1. 如何安装软件：

  1. 下载 dmg 格式文件安装，可从 soft.macx.cn 下载，点击 dmg 文件安装

  2. 软件安装后从何处调出： Leopard 中 Finder->应用程序
    Lion 中 launchPad（程序启动器）中调出，可用三指翻页

  1. 浏览器的使用

  1. 使用浏览器上网：屏幕下方的 safari 就是 Mac OS 中的浏览器

  2. 浏览器退出全屏：按两次 Esc 键

  3. 浏览器加书签：按地址栏之前的加号图标

  4. 浏览器新建标签页：右键点击浏览器 Title bar，自定工具栏，将“新建标签”按钮拖入工具栏

  1. 如何使用 office：office 文件在 mac 中可以打开和编辑，如需要使用 office，可下载 office dmg 包安装，需要购买 license

  2. Mac OS 使用 Windows 系统：

  1. 使用 Mac OS 自带的 Bootcamp（Lion 只能安装 Window7）

  2. 远程桌面：可从下载 RDC 的 dmg 包安装

  3. 使用 VirtualBox 安装 Windows 虚拟机

4．系统设置

  1. 修改密码及 AppleID： Lion：系统偏好设置 ->用户与组
    leopard：系统偏好设置 ->帐户

  2. 用户数据存放位置： /User/用户名/

  3. 修改字体大小

  1. 设置当前应用字体大小：当前应用显示在左上角，选择其中的偏好设置

  2. 设置系统字体，需下载 tinkertool 工具，详见： [http://bresink.de/osx/TinkerTool.html

](http://bresink.de/osx/TinkerTool.html)

  1. 与其它机器共享数据
    系统偏好设置 ->共享，选择文件共享, 点击“选项”按钮，勾选 Samba（即网上邻居），勾选一用户，使用下方加号，添加需要共享的目录

  2. 输入法改为 Ctrl+ 空格切换
    点击项端工具条右上角的输入法图标 ->打开语言与文偏好设置 ->键盘快捷键，选中左侧的“spotlight”，去掉 Ctrl+ 空格的勾选；选中左侧的“键盘与文本输入”，双击右侧的“选择上一个输入源”右侧的快捷键，按 Ctrl+ 空格，即可修改切换输入法的快捷键。详见： http://www.crifan.com/change_shortcut_of_input_method_in_mac_to_ctrl_space/

5．建立 Iphone 开发环境

  1. 下载 Xcode
    Xcode 是 mac 下的开发环境（非开源），用于开发 mac os，iphone, ipad 软件
    Xcode 不同版本对应不同 Mac OS 版本，下载相应 dmg 安装后即可使用，版本对应详见： [http://elf8848.iteye.com/blog/1366101

](http://elf8848.iteye.com/blog/1366101)
